# CLI Command History

## `git checkout -b feat/big-bang-integration`

Creates a new Git branch named "feat/big-bang-integration" and switches to it.

## `dev build`

Builds the development environment.

## `ct dump 30`

Dumps the first 30 lines of the ct output.

Dumps the first 2 lines of the ct output.
