# CLI Command History

## `git log --oneline -3`

Displays the last 3 commit messages in a concise, one-line format.

## `git add -A`

Stages all changes in the repository, including new, modified, and deleted files.

## `git commit -m "feat: default command is to fetch the last 20 commands"`

Commits changes with a message, in this case describing a feature for fetching the last 20 commands.

## `git push origin feat/default-command-fetch-last-20-commands`

Pushes the committed changes to the remote repository's specified branch.

## `gits`

## `git fetch origin master:master`

Fetches the master branch from the origin remote repository and updates the local master branch.

## `gitb`

This command is not a valid Git command, possibly a typo.

## `git checkout master`

Switches to the master branch in the local repository.

## `git checkout -b feat/markdown-for-past-cli-commands`

Creates a new branch named feat/markdown-for-past-cli-commands and switches to it.

## `go get github.com/sashabaranov/go-openai\`

Downloads and installs the go-openai package using the Go package manager.

## `make dev`

Runs the development target as defined in the Makefile.

## `echo $CLIDUMP_OPENAI_KEY`

Prints the value of the environment variable CLIDUMP_OPENAI_KEY.
